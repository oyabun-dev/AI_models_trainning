# Prérequis pour le Modèle DINOv2

Pour pouvoir exécuter le script `app.py` basé sur le modèle d'intelligence artificielle DINOv2 (de Meta Research), vous devez installer les dépendances nécessaires.

## 📦 Installation

Ouvrez votre terminal et exécutez la commande suivante pour installer la bibliothèque `transformers` directement depuis le dépôt GitHub d'Hugging Face :

```bash
pip install git+https://github.com/huggingface/transformers.git
```
